# Quick Start

Ptero Manager Needs Some API Key's To Work Properly. You Can Get All The API Key's From Your Pterodactyl Panel.

### Things You'll Need To Get Started!

- Pterodactyl Panel API
- Pterodactyl Panel Client API
- Nest Egg's Details
- Welcome Channel, Get Started Channel
- Client Role
- Channel Create Category
- SMTP (Optional)
- Top.GG Vote (Optional)
